class app.backend.grade_monitor.GradeMonitor(grades_list, ignore_ects=False)¶

Bases: object

Purpose of the class is to collect, manage and operate on grade data from the database.

calculate_subject_average(subject_name)¶

Method calculates average grade for specified subject :type subject_name: str :param subject_name: Name of a subject which average is calculated :rtype: float :return: Average subject grade as a float value

calculate_subject_regular_average(subject_name)¶

Method calculates subject average in a regular grading system with one module per subject with grade weights :type subject_name: str :param subject_name: Name of a subject which average is calculated :rtype: float :return: Average subject grade as a float value

calculate_subject_type_average(subject_name)¶

Method returns averages for a specified subject for each type of grade in a dictionary form with weight of type attached :type subject_name: str :param subject_name: Name of a subject of which grade type averages are calculated :rtype: dict[GradeType, tuple[float, float]] :return: Dictionary of a subject grade type and a tuple containing its average and weight

calculate_total_grade_average()¶

Method calculates average grade of all subjects. :rtype: float :return: Total average grade as float value

fill_monitor_tables(grades_list)¶

Method fills monitor’s tables with data fetched from database. :type grades_list: list[tuple[float, str, int, float, int, int]] :param grades_list: Table of grades data fetched from database :rtype: None :return: Nothing

grade_counts(subject_names=None)¶

Method counts quantity of each grade, possible filtering by subject name :type subject_names: :param subject_names: Optional parameter - names of subjects to count grades from :rtype: dict[float, int] :return: Dictionary containing counts of each grade
